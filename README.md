# JS-FINDER
JS-FINDER

JS-FINDER is a lightweight browser-based JavaScript discovery tool built for educational cybersecurity research.It extracts external JavaScript files from a target domain and highlights filenames that may contain sensitive information.

How It Works

Enter a target domain.

The tool fetches page source.

Extracts linked .js files.

Flags filenames containing high-risk keywords.

Displays results inside styled terminal panels.
