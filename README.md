# JS-FINDER
JS-FINDER

JS-FINDER is a lightweight browser-based JavaScript discovery tool built for educational cybersecurity research.It extracts external JavaScript files from a target domain and highlights filenames that may contain sensitive information.

# How It Works

  1 Enter a target domain.

  2 The tool fetches page source.

  3 Extracts linked .js files.

  4 Flags filenames containing high-risk keywords.

  5 Displays results inside styled terminal panels.

  6 Click on urls to inspect the js and find the sensitive info
